# MatchTheTerror
This is a GUI based memory puzzle game developed using Java Swing and it follows an event-driven programming model.The objective of this game is to pair historical figures(Dictators) with their respective countries of origin. This game features a dark theme, custom animations, a scoring timer, and a responsive grid layout.

## Running The Application

```sh
javac src/MatchTheTerror.java
java -cp src MatchTheTerror
```
